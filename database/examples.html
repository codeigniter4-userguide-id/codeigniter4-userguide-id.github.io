Mulai Cepat: Contoh Penggunaan

Halaman berikut berisi contoh kode yang menunjukkan bagaimana kelas database digunakan. Untuk detail lengkap, silakan baca halaman individual yang menjelaskan setiap fungsi.

Catatan

CodeIgniter tidak mendukung titik (.) dalam nama tabel dan kolom. Sejak v4.5.0, nama database dengan titik didukung.

Menginisialisasi Kelas Database

Kode berikut memuat dan menginisialisasi kelas database berdasarkan pengaturan konfigurasi Anda:

<?php

$db = \Config\Database::connect();

Setelah dimuat, kelas siap digunakan seperti yang dijelaskan di bawah ini.

Catatan

Jika semua halaman Anda memerlukan akses database, Anda dapat terhubung secara otomatis. Lihat halaman Menghubungkan ke Database untuk detailnya.

Kueri Standar Dengan Banyak Hasil (Versi Objek)

<?php

$query   = $db->query('SELECT name, title, email FROM my_table');
$results = $query->getResult();

foreach ($results as $row) {
    echo $row->title;
    echo $row->name;
    echo $row->email;
}

echo 'Total Results: ' . count($results);
Fungsi getResult() di atas mengembalikan array objek.
Contoh: $row->title

Kueri Standar Dengan Banyak Hasil (Versi Array)

<?php

$query   = $db->query('SELECT name, title, email FROM my_table');
$results = $query->getResultArray();

foreach ($results as $row) {
    echo $row['title'];
    echo $row['name'];
    echo $row['email'];
}
Fungsi getResultArray() di atas mengembalikan array indeks array standar.
Contoh: $row['title']

Kueri Standar Dengan Hasil Tunggal

<?php

$query = $db->query('SELECT name FROM my_table LIMIT 1');
$row   = $query->getRow();
echo $row->name;

Fungsi getRow() di atas mengembalikan objek. Contoh: $row->name

Kueri Standar Dengan Hasil Tunggal (Versi Array)

<?php

$query = $db->query('SELECT name FROM my_table LIMIT 1');
$row   = $query->getRowArray();
echo $row['name'];

Fungsi getRowArray() di atas mengembalikan array. Contoh: $row['name'].

Insert Standar

<?php

$sql = 'INSERT INTO mytable (title, name) VALUES (' . $db->escape($title) . ', ' . $db->escape($name) . ')';
$db->query($sql);
echo $db->affectedRows();

Kueri Query Builder

Query Builder memberi Anda cara yang disederhanakan untuk mengambil data:

<?php

$query = $db->table('table_name')->get();

foreach ($query->getResult() as $row) {
    echo $row->title;
}

Fungsi get() di atas mengambil semua hasil dari tabel yang diberikan. Kelas Query Builder berisi kumpulan lengkap fungsi untuk bekerja dengan data.

Insert Query Builder

<?php

$data = [
    'title' => $title,
    'name'  => $name,
    'date'  => $date,
];

$db->table('mytable')->insert($data);
// Produces: INSERT INTO mytable (title, name, date) VALUES ('{$title}', '{$name}', '{$date}')
