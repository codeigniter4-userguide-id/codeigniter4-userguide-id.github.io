Metadata Tabel

Fungsi-fungsi ini memungkinkan Anda mengambil informasi tabel.

Daftar Tabel di Database Anda

$db->listTables()

Mengembalikan array yang berisi nama semua tabel di database yang sedang Anda sambungkan. Contoh:

<?php

$db = db_connect();

$tables = $db->listTables();

foreach ($tables as $table) {
    echo $table;
}

Catatan

Beberapa driver memiliki tabel sistem tambahan yang dikecualikan dari pengembalian ini.

Menentukan Apakah Tabel Ada

$db->tableExists()

Terkadang berguna untuk mengetahui apakah tabel tertentu ada sebelum menjalankan operasi di atasnya. Mengembalikan boolean true/false. Contoh penggunaan:

<?php

$db = db_connect();

if ($db->tableExists('table_name')) {
    // some code...
}

Catatan

Ganti table_name dengan nama tabel yang Anda cari.

Metadata Field

Daftar Field dalam Tabel

$db->getFieldNames()

Mengembalikan array yang berisi nama field. Kueri ini dapat dipanggil dengan dua cara:

  1. Anda dapat memberikan nama tabel dan memanggilnya dari objek $db:

    <?php

$db = db_connect();

$fields = $db->getFieldNames('table_name');

foreach ($fields as $field) {
    echo $field;
}

  2. Anda dapat mengumpulkan nama field yang terkait dengan kueri apa pun yang Anda jalankan dengan memanggil fungsi dari objek hasil kueri Anda:

    <?php

$db = db_connect();

$query = $db->query('SELECT * FROM some_table');

foreach ($query->getFieldNames() as $field) {
    echo $field;
}

Menentukan Apakah Field Ada dalam Tabel

$db->fieldExists()

Terkadang berguna untuk mengetahui apakah field tertentu ada sebelum melakukan tindakan. Mengembalikan boolean true/false. Contoh penggunaan:

<?php

$db = db_connect();

if ($db->fieldExists('field_name', 'table_name')) {
    // some code...
}

Catatan

Ganti field_name dan table_name dengan nama kolom dan nama tabel yang Anda cari.

Mengambil Metadata Field

$db->getFieldData()

Mengembalikan array objek yang berisi informasi field.

Terkadang berguna untuk mengumpulkan nama field atau metadata lainnya, seperti tipe kolom, panjang maksimum, dll.

Catatan

Tidak semua database menyediakan metadata.

Contoh penggunaan:

<?php

$db = db_connect();

$fields = $db->getFieldData('table_name');

foreach ($fields as $field) {
    echo $field->name;
    echo $field->type;
    echo $field->max_length;
    echo $field->primary_key;
}

Data berikut tersedia dari fungsi ini jika didukung oleh database Anda:

  • name - nama kolom

  • type - tipe kolom

  • max_length - panjang maksimum kolom

  • nullable - boolean true jika kolom bisa null, jika tidak boolean false

  • default - nilai default

  • primary_key - integer 1 jika kolom adalah primary key (semua integer 1, bahkan jika ada beberapa primary key), jika tidak integer 0 (Field ini saat ini hanya tersedia untuk MySQLi dan SQLite3)

Catatan

Sejak v4.4.0, SQLSRV mendukung nullable.

$query->getFieldData()

Jika Anda sudah menjalankan kueri, Anda dapat menggunakan objek hasil alih-alih memberikan nama tabel:

<?php

$db = db_connect();

$query  = $db->query('YOUR QUERY');
$fields = $query->getFieldData();

Catatan

Data yang dikembalikan berbeda dari data dari $db->getFieldData(). Jika Anda tidak bisa mendapatkan data yang Anda butuhkan, gunakan $db->getFieldData().

Daftar Indeks dalam Tabel

$db->getIndexData()

Mengembalikan array objek yang berisi informasi indeks.

Contoh penggunaan:

<?php

$db = db_connect();

$keys = $db->getIndexData('table_name');

foreach ($keys as $key) {
    echo $key->name;
    echo $key->type;
    echo $key->fields; // array of field names
}

Tipe kunci mungkin unik untuk database yang Anda gunakan. Misalnya, MySQL akan mengembalikan salah satu dari primary, fulltext, spatial, index atau unique untuk setiap kunci yang terkait dengan tabel.

SQLite3 mengembalikan indeks pseudo bernama PRIMARY. Tapi itu adalah indeks khusus, dan Anda tidak bisa menggunakannya dalam perintah SQL Anda.

$db->getForeignKeyData()

Mengembalikan array objek yang berisi informasi foreign key.

Contoh penggunaan:

<?php

$db = db_connect();

$keys = $db->getForeignKeyData('table_name');

foreach ($keys as $key => $object) {
    echo $key === $object->constraint_name;
    echo $object->constraint_name;
    echo $object->table_name;
    echo $object->column_name[0]; // array
    echo $object->foreign_table_name;
    echo $object->foreign_column_name[0]; // array
    echo $object->on_delete;
    echo $object->on_update;
    echo $object->match;
}

Foreign key menggunakan konvensi penamaan tableprefix_table_column1_column2_foreign. Oracle menggunakan sufiks yang sedikit berbeda yaitu _fk.
