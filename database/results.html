Menghasilkan Query Results

Ada beberapa cara untuk menghasilkan query results:

  • Result Arrays

    • getResult()

      • Mendapatkan Array stdClass

      • Mendapatkan Array of Array

      • Mendapatkan Array of Custom Object

    • getResultArray()

  • Result Rows

    • getRow()

    • getRowArray()

    • getUnbufferedRow()

  • Custom Result Objects

    • getCustomResultObject()

    • getCustomRowObject()

  • Result Helper Methods

    • getFieldCount()

    • getFieldNames()

    • getNumRows()

    • freeResult()

    • dataSeek()

  • Referensi Class

Result Arrays

getResult()

Metode ini mengembalikan query result sebagai array objects, atau array kosong jika gagal.

Mendapatkan Array stdClass

Biasanya Anda akan menggunakan ini dalam foreach loop, seperti ini:

<?php

$query = $db->query('YOUR QUERY');

foreach ($query->getResult() as $row) {
    echo $row->title;
    echo $row->name;
    echo $row->body;
}

Metode di atas adalah alias dari CodeIgniter\Database\BaseResult::getResultObject().

Mendapatkan Array of Array

Anda dapat meneruskan string 'array' jika Anda ingin mendapatkan results Anda sebagai array of arrays:

<?php

$query = $db->query('YOUR QUERY');

foreach ($query->getResult('array') as $row) {
    echo $row['title'];
    echo $row['name'];
    echo $row['body'];
}

Penggunaan di atas adalah alias dari getResultArray().

Mendapatkan Array of Custom Object

Anda juga dapat meneruskan string ke getResult() yang merepresentasikan class untuk menginisialisasi setiap result object

<?php

$query = $db->query('SELECT * FROM users;');

foreach ($query->getResult(\App\Entities\User::class) as $user) {
    echo $user->name;          // access attributes
    echo $user->reverseName(); // or methods defined on the 'User' class
}

Metode di atas adalah alias dari getCustomResultObject().

getResultArray()

Metode ini mengembalikan query result sebagai pure array, atau empty array ketika tidak ada result yang dihasilkan. Biasanya Anda akan menggunakan ini dalam foreach loop, seperti ini:

<?php

$query = $db->query('YOUR QUERY');

foreach ($query->getResultArray() as $row) {
    echo $row['title'];
    echo $row['name'];
    echo $row['body'];
}

Result Rows

getRow()

Metode ini mengembalikan single result row. Jika query Anda memiliki lebih dari satu row, ini hanya mengembalikan first row. Result dikembalikan sebagai object. Berikut contoh penggunaannya:

<?php

$query = $db->query('YOUR QUERY');

$row = $query->getRow();

if (isset($row)) {
    echo $row->title;
    echo $row->name;
    echo $row->body;
}

Jika Anda ingin row tertentu dikembalikan, Anda dapat mengirimkan nomor row sebagai digit di parameter pertama:

<?php

$row = $query->getRow(5);

Anda juga dapat menambahkan parameter String kedua, yaitu nama class untuk menginisialisasi row tersebut:

<?php

$query = $db->query('SELECT * FROM users LIMIT 1;');
$row   = $query->getRow(0, \App\Entities\User::class);

echo $row->name;           // access attributes
echo $row->reverse_name(); // or methods defined on the 'User' class

getRowArray()

Identik dengan metode row() di atas, kecuali mengembalikan array. Contoh:

<?php

$query = $db->query('YOUR QUERY');

$row = $query->getRowArray();

if (isset($row)) {
    echo $row['title'];
    echo $row['name'];
    echo $row['body'];
}

Jika Anda ingin row tertentu dikembalikan, Anda dapat mengirimkan nomor row sebagai digit di parameter pertama:

<?php

$row = $query->getRowArray(5);

Selain itu, Anda dapat berjalan maju/mundur/pertama/terakhir melalui results Anda menggunakan variasi ini:

$row = $query->getFirstRow()
$row = $query->getLastRow()
$row = $query->getNextRow()
$row = $query->getPreviousRow()

Secara default, mereka mengembalikan object kecuali Anda menempatkan kata "array" di parameter:

$row = $query->getFirstRow('array')
$row = $query->getLastRow('array')
$row = $query->getNextRow('array')
$row = $query->getPreviousRow('array')

Catatan

Semua metode di atas akan memuat seluruh result ke dalam memory (prefetching). Gunakan getUnbufferedRow() untuk memproses large result sets.

getUnbufferedRow()

Metode ini mengembalikan single result row tanpa prefetching seluruh result di memory seperti yang dilakukan row(). Jika query Anda memiliki lebih dari satu row, ini mengembalikan current row dan memindahkan internal data pointer ke depan.

<?php

$query = $db->query('YOUR QUERY');

while ($row = $query->getUnbufferedRow()) {
    echo $row->title;
    echo $row->name;
    echo $row->body;
}

Untuk penggunaan dengan MySQLi, Anda dapat mengatur result mode MySQLi ke MYSQLI_USE_RESULT untuk penghematan memory maksimum. Penggunaan ini tidak umumnya direkomendasikan tetapi dapat bermanfaat dalam beberapa keadaan seperti menulis large queries ke csv. Jika Anda mengubah result mode perhatikan tradeoffs yang terkait dengannya.

<?php

$db->resultMode = MYSQLI_USE_RESULT; // for unbuffered results

$query = $db->query('YOUR QUERY');

$file = new \CodeIgniter\Files\File(WRITEPATH . 'data.csv');

$csv = $file->openFile('w');

while ($row = $query->getUnbufferedRow('array')) {
    $csv->fputcsv($row);
}

$db->resultMode = MYSQLI_STORE_RESULT; // return to default mode

Catatan

Saat menggunakan MYSQLI_USE_RESULT semua subsequent calls pada connection yang sama akan menghasilkan error sampai semua records telah diambil atau panggilan freeResult() telah dilakukan. Metode getNumRows() hanya akan mengembalikan number of rows berdasarkan current position dari data pointer. Tabel MyISAM akan tetap locked sampai semua records telah diambil atau panggilan freeResult() telah dilakukan.

Anda dapat secara opsional meneruskan 'object' (default) atau 'array' untuk menentukan type nilai yang dikembalikan:

<?php

$query->getUnbufferedRow();         // object
$query->getUnbufferedRow('object'); // object
$query->getUnbufferedRow('array');  // associative array

Custom Result Objects

Anda dapat meminta results dikembalikan sebagai instance dari custom class bukan stdClass atau array, seperti yang diizinkan oleh metode getResult() dan getResultArray(). Jika class belum dimuat ke dalam memory, Autoloader akan mencoba memuatnya. Object akan memiliki semua values yang dikembalikan dari database yang diatur sebagai properties. Jika ini telah dideklarasikan dan non-public maka Anda harus menyediakan metode __set() untuk memungkinkan mereka diatur.

Contoh:

<?php

namespace App\Entities;

class User
{
    public $id;
    public $email;
    public $username;

    protected $lastLogin;

    public function lastLogin($format)
    {
        return $this->lastLogin->format($format);
    }

    public function __set($name, $value)
    {
        if ($name === 'lastLogin') {
            $this->lastLogin = DateTime::createFromFormat('!U', $value);
        }
    }

    public function __get($name)
    {
        if (isset($this->{$name})) {
            return $this->{$name};
        }
    }
}

Selain dua metode yang tercantum di bawah ini, metode berikut juga dapat mengambil class name untuk mengembalikan results sebagai: getFirstRow(), getLastRow(), getNextRow(), dan getPreviousRow().

getCustomResultObject()

Mengembalikan seluruh result set sebagai array of instances dari class yang diminta. Satu-satunya parameter adalah nama class untuk diinisialisasi.

Contoh:

<?php

$query = $db->query('YOUR QUERY');

$rows = $query->getCustomResultObject(\App\Entities\User::class);

foreach ($rows as $row) {
    echo $row->id;
    echo $row->email;
    echo $row->lastLogin('Y-m-d');
}

getCustomRowObject()

Mengembalikan single row dari query results Anda. Parameter pertama adalah row number dari results. Parameter kedua adalah class name untuk diinisialisasi.

Contoh:

<?php

$query = $db->query('YOUR QUERY');

$row = $query->getCustomRowObject(0, \App\Entities\User::class);

if (isset($row)) {
    echo $row->email;              // access attributes
    echo $row->lastLogin('Y-m-d'); // access class methods
}

Anda juga dapat menggunakan metode getRow() dengan cara yang persis sama.

Contoh:

<?php

$row = $query->getRow(0, \App\Entities\User::class);

Result Helper Methods

getFieldCount()

Number of FIELDS (columns) yang dikembalikan oleh query. Pastikan untuk memanggil metode menggunakan query result object Anda:

<?php

$query = $db->query('SELECT * FROM my_table');

echo $query->getFieldCount();

getFieldNames()

Mengembalikan array dengan names of the FIELDS (columns) yang dikembalikan oleh query. Pastikan untuk memanggil metode menggunakan query result object Anda:

<?php

$query = $db->query('SELECT * FROM my_table');

echo $query->getFieldNames();

getNumRows()

Number of records yang dikembalikan oleh query. Pastikan untuk memanggil metode menggunakan query result object Anda:

<?php

$query = $db->query('SELECT * FROM my_table');

echo $query->getNumRows();

Catatan

Karena SQLite3 tidak memiliki metode yang efisien untuk mengembalikan record count, CodeIgniter akan mengambil dan mem-buffer record query result secara internal dan mengembalikan count dari resulting record array, yang bisa jadi tidak efisien.

freeResult()

Ini membebaskan memory yang terkait dengan result dan menghapus result resource ID. Biasanya PHP membebaskan memory-nya secara otomatis di akhir script execution. Namun, jika Anda menjalankan banyak queries dalam script tertentu, Anda mungkin ingin membebaskan result setelah setiap query result telah dihasilkan untuk mengurangi memory consumption.

Contoh:

<?php

$query = $thisdb->query('SELECT title FROM my_table');

foreach ($query->getResult() as $row) {
    echo $row->title;
}

$query->freeResult(); // The $query result object will no longer be available

$query2 = $db->query('SELECT name FROM some_table');

$row = $query2->getRow();
echo $row->name;
$query2->freeResult(); // The $query2 result object will no longer be available

dataSeek()

Metode ini mengatur internal pointer untuk next result row yang akan diambil. Ini hanya berguna dalam kombinasi dengan getUnbufferedRow().

Ini menerima positive integer value, yang defaultnya adalah 0 dan mengembalikan true jika berhasil atau false jika gagal.

<?php

$query = $db->query('SELECT `field_name` FROM `table_name`');
$query->dataSeek(5); // Skip the first 5 rows
$row = $query->getUnbufferedRow();

Catatan

Tidak semua database drivers mendukung feature ini dan akan mengembalikan false. Yang paling penting - Anda tidak akan dapat menggunakannya dengan PDO.
