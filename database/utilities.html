Utilitas Database

Class Utilitas Database berisi metode-metode yang membantu Anda mengelola database Anda.

  • Menginisialisasi Class Utilitas

  • Menggunakan Utilitas Database

    • Mengambil Daftar Nama Database

    • Mengoptimalkan Database

Menginisialisasi Class Utilitas

Muat Class Utilitas sebagai berikut:

$dbutil = \Config\Database::utils();

Anda juga dapat meneruskan grup database lain ke pemuat Utilitas DB, jika database yang ingin Anda kelola bukan yang default:

$dbutil = \Config\Database::utils('group_name');

Pada contoh di atas, kita meneruskan nama grup database sebagai parameter pertama.

Menggunakan Utilitas Database

Mengambil Daftar Nama Database

Mengembalikan array nama database:

$dbutil = \Config\Database::utils();

$dbs = $dbutil->listDatabases();

foreach ($dbs as $db) {
    echo $db;
}

Menentukan Apakah Database Ada

Terkadang sangat membantu untuk mengetahui apakah database tertentu ada. Mengembalikan boolean true/false. Contoh penggunaan:

$dbutil = \Config\Database::utils();

if ($dbutil->databaseExists('database_name')) {
    // some code...
}

Catatan

Ganti database_name dengan nama database yang Anda cari. Metode ini peka huruf besar/kecil.

Mengoptimalkan Tabel

Memungkinkan Anda untuk mengoptimalkan tabel menggunakan nama tabel yang ditentukan dalam parameter pertama. Mengembalikan true/false berdasarkan keberhasilan atau kegagalan:

$dbutil = \Config\Database::utils();

if ($dbutil->optimizeTable('table_name')) {
    echo 'Success!';
}

Catatan

Tidak semua platform database mendukung optimasi tabel. Ini kebanyakan untuk digunakan dengan MySQL.

Mengoptimalkan Database

Memungkinkan Anda untuk mengoptimalkan database yang saat ini terhubung dengan class DB Anda. Mengembalikan array yang berisi pesan status DB atau false jika gagal:

$dbutil = \Config\Database::utils();

$result = $dbutil->optimizeDatabase();

if ($result !== false) {
    print_r($result);
}

Catatan

Tidak semua platform database mendukung optimasi database. Ini kebanyakan untuk digunakan dengan MySQL.

Export Hasil Kueri sebagai CSV File

Memungkinkan Anda untuk menghasilkan CSV file dari hasil kueri. Parameter pertama dari metode harus berisi objek hasil dari kueri Anda. Contoh:

$db     = db_connect();
$dbutil = \Config\Database::utils();

$query = $db->query('SELECT * FROM mytable');

echo $dbutil->getCSVFromResult($query);

Parameter kedua, ketiga, dan keempat memungkinkan Anda untuk mengatur karakter pemisah, baris baru, dan pembatas. Secara default koma digunakan sebagai pemisah, "\n" digunakan sebagai baris baru, dan tanda kutip ganda digunakan sebagai pembatas. Contoh:

$db     = db_connect();
$dbutil = \Config\Database::utils();

$query = $db->query('SELECT * FROM mytable');

$delimiter = ',';
$newline   = "\r\n";
$enclosure = '"';

echo $dbutil->getCSVFromResult($query, $delimiter, $newline, $enclosure);

Penting

Metode ini TIDAK akan menulis CSV file untuk Anda. Ini hanya membuat tata letak CSV. Jika Anda perlu menulis file-nya gunakan helper write_file().

Export Hasil Kueri sebagai XML Document

Memungkinkan Anda untuk menghasilkan XML document dari hasil kueri. Parameter pertama mengharapkan objek hasil kueri, yang kedua dapat berisi array opsional parameter konfigurasi. Contoh:

<?php

$db     = db_connect();
$dbutil = \Config\Database::utils();

$query = $db->query('SELECT * FROM mytable');

$config = [
    'root'    => 'root',
    'element' => 'element',
    'newline' => "\n",
    'tab'     => "\t",
];

echo $dbutil->getXMLFromResult($query, $config);

dan itu akan mendapatkan hasil xml berikut ketika mytable memiliki kolom id dan name:

<root>
    <element>
        <id>1</id>
        <name>bar</name>
    </element>
</root>

Penting

Metode ini TIDAK akan menulis XML document untuk Anda. Ini hanya membuat tata letak XML. Jika Anda perlu menulis file-nya gunakan helper write_file().
