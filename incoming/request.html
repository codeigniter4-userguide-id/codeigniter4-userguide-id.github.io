Class Request

Class request adalah representasi berorientasi objek dari sebuah permintaan HTTP. Ini dimaksudkan untuk berfungsi baik untuk permintaan masuk, seperti permintaan ke aplikasi dari browser, maupun permintaan keluar, seperti yang akan digunakan untuk mengirim permintaan dari aplikasi ke aplikasi pihak ketiga.

Class ini menyediakan fungsionalitas umum yang keduanya butuhkan, tetapi kedua kasus memiliki class kustom yang memperluas dari class Request untuk menambahkan fungsionalitas spesifik. Dalam praktiknya, Anda perlu menggunakan class-class ini.

Lihat dokumentasi untuk Class IncomingRequest dan Class CURLRequest untuk detail penggunaan lebih lanjut.

Referensi Class

class CodeIgniter\HTTP\Request
getIPAddress()
Kembali:

Alamat IP pengguna, jika dapat dideteksi. Jika alamat IP bukan alamat IP yang valid, maka akan mengembalikan 0.0.0.0.

Return type:

string

Mengembalikan alamat IP untuk pengguna saat ini. Jika alamat IP tidak valid, method akan mengembalikan 0.0.0.0:

<?php

echo $request->getIPAddress();

Penting

Method ini memperhitungkan pengaturan Config\App::$proxyIPs dan akan mengembalikan alamat IP klien yang dilaporkan oleh header HTTP untuk alamat IP yang diizinkan.

isValidIP($ip[, $which = ''])

Ditinggalkan sejak versi 4.0.5: Gunakan Validation sebagai gantinya.

Penting

Method ini sudah usang. Ini akan dihapus di rilis mendatang.

Parameter:

  • $ip (string) -- Alamat IP

  • $which (string) -- Protokol IP (ipv4 atau ipv6)

Kembali:

true jika alamat valid, false jika tidak

Return type:

bool

Mengambil alamat IP sebagai input dan mengembalikan true atau false (boolean) tergantung apakah itu valid atau tidak.

Catatan

Method $request->getIPAddress() di atas secara otomatis memvalidasi alamat IP.

<?php

if (! $request->isValidIP($ip)) {
    echo 'Not Valid';
} else {
    echo 'Valid';
}

Menerima parameter string kedua opsional berupa ipv4 atau ipv6 untuk menentukan format IP. Defaultnya memeriksa kedua format.

getMethod()
Kembali:

Method permintaan HTTP

Return type:

string

Mengembalikan $_SERVER['REQUEST_METHOD'].

<?php

echo $request->getMethod(); // Outputs: POST
setMethod($method)

Ditinggalkan sejak versi 4.0.5: Gunakan CodeIgniter\HTTP\Request::withMethod() sebagai gantinya.

Parameter:

  • $method (string) -- Mengatur method permintaan. Digunakan saat memalsukan permintaan.

Kembali:

Permintaan ini

Return type:

Request

withMethod($method)

Baru pada versi 4.0.5.

Parameter:

  • $method (string) -- Mengatur method permintaan.

Kembali:

Instance permintaan baru

Return type:

Request

getServer([$index = null[, $filter = null[, $flags = null]]])
Parameter:

  • $index (mixed) -- Nama nilai

  • $filter (int) -- Jenis filter yang akan diterapkan. Daftar filter dapat ditemukan di manual PHP.

  • $flags (int|array) -- Flag yang akan diterapkan. Daftar flag dapat ditemukan di manual PHP.

Kembali:

Nilai item $_SERVER jika ditemukan, null jika tidak

Return type:

mixed

Method ini identik dengan method getPost(), getGet() dan getCookie() dari Class IncomingRequest, hanya saja ini mengambil data server ($_SERVER):

<?php

$request->getServer('some_data');

Untuk mengembalikan array dari beberapa nilai $_SERVER, berikan semua kunci yang diperlukan sebagai array.

<?php

$request->getServer(['SERVER_PROTOCOL', 'REQUEST_URI']);
getEnv([$index = null[, $filter = null[, $flags = null]]])

Ditinggalkan sejak versi 4.4.4: Method ini tidak berfungsi sejak awal. Gunakan env() sebagai gantinya.

Parameter:

  • $index (mixed) -- Nama nilai

  • $filter (int) -- Jenis filter yang akan diterapkan. Daftar filter dapat ditemukan di manual PHP.

  • $flags (int|array) -- Flag yang akan diterapkan. Daftar flag dapat ditemukan di manual PHP.

Kembali:

Nilai item $_ENV jika ditemukan, null jika tidak

Return type:

mixed

Method ini identik dengan method getPost(), getGet() dan getCookie() dari Class IncomingRequest, hanya saja ini mengambil data env ($_ENV):

<?php

$request->getEnv('some_data');

Untuk mengembalikan array dari beberapa nilai $_ENV, berikan semua kunci yang diperlukan sebagai array.

<?php

$request->getEnv(['CI_ENVIRONMENT', 'S3_BUCKET']);
setGlobal($method, $value)
Parameter:

  • $method (string) -- Nama method

  • $value (mixed) -- Data yang akan ditambahkan

Kembali:

Permintaan ini

Return type:

Request

Memungkinkan pengaturan nilai global PHP secara manual, seperti $_GET, $_POST, dll.

fetchGlobal($method[, $index = null[, $filter = null[, $flags = null]]])
Parameter:

  • $method (string) -- Konstanta filter input

  • $index (mixed) -- Nama nilai

  • $filter (int) -- Jenis filter yang akan diterapkan. Daftar filter dapat ditemukan di manual PHP.

  • $flags (int|array) -- Flag yang akan diterapkan. Daftar flag dapat ditemukan di manual PHP.

Return type:

mixed

Mengambil satu atau lebih item dari global, seperti cookie, get, post, dll. Dapat secara opsional memfilter input saat Anda mengambilnya dengan meneruskan filter.
