HTML Table Class

Class Table menyediakan metode yang memungkinkan Anda untuk secara otomatis menghasilkan tabel HTML dari array atau hasil set database.

  • Menggunakan Class Table

    • Menginisialisasi Class

    • Contoh

    • Mengubah Tampilan Tabel Anda

    • Menyinkronkan Baris dengan Judul

  • Referensi Class

Menggunakan Class Table

Menginisialisasi Class

Class Table tidak disediakan sebagai service, dan harus diinstansiasi "secara normal", misalnya:

<?php

$table = new \CodeIgniter\View\Table();

Contoh

Berikut adalah contoh yang menunjukkan bagaimana Anda dapat membuat tabel dari array multi-dimensi. Perhatikan bahwa indeks array pertama akan menjadi judul tabel (atau Anda dapat mengatur judul Anda sendiri menggunakan metode setHeading() yang dijelaskan dalam referensi fungsi di bawah).

<?php

$table = new \CodeIgniter\View\Table();

$data = [
    ['Name', 'Color', 'Size'],
    ['Fred', 'Blue', 'Small'],
    ['Mary', 'Red', 'Large'],
    ['John', 'Green', 'Medium'],
];

echo $table->generate($data);

Berikut adalah contoh tabel yang dibuat dari hasil query database. Class tabel akan secara otomatis menghasilkan judul berdasarkan nama tabel (atau Anda dapat mengatur judul Anda sendiri menggunakan metode setHeading() yang dijelaskan dalam referensi class di bawah).

<?php

$table = new \CodeIgniter\View\Table();

$query = $db->query('SELECT * FROM my_table');

echo $table->generate($query);

Berikut adalah contoh yang menunjukkan bagaimana Anda dapat membuat tabel menggunakan parameter diskrit:

<?php

$table = new \CodeIgniter\View\Table();

$table->setHeading('Name', 'Color', 'Size');

$table->addRow('Fred', 'Blue', 'Small');
$table->addRow('Mary', 'Red', 'Large');
$table->addRow('John', 'Green', 'Medium');

echo $table->generate();

Berikut adalah contoh yang sama, kecuali alih-alih parameter individual, array digunakan:

<?php

$table = new \CodeIgniter\View\Table();

$table->setHeading(['Name', 'Color', 'Size']);

$table->addRow(['Fred', 'Blue', 'Small']);
$table->addRow(['Mary', 'Red', 'Large']);
$table->addRow(['John', 'Green', 'Medium']);

echo $table->generate();

Mengubah Tampilan Tabel Anda

Class Table memungkinkan Anda untuk mengatur template tabel yang dengannya Anda dapat menentukan desain layout Anda. Berikut adalah prototipe template:

<?php

$template = [
    'table_open' => '<table border="0" cellpadding="4" cellspacing="0">',

    'thead_open'  => '<thead>',
    'thead_close' => '</thead>',

    'heading_row_start'  => '<tr>',
    'heading_row_end'    => '</tr>',
    'heading_cell_start' => '<th>',
    'heading_cell_end'   => '</th>',

    'tfoot_open'  => '<tfoot>',
    'tfoot_close' => '</tfoot>',

    'footing_row_start'  => '<tr>',
    'footing_row_end'    => '</tr>',
    'footing_cell_start' => '<td>',
    'footing_cell_end'   => '</td>',

    'tbody_open'  => '<tbody>',
    'tbody_close' => '</tbody>',

    'row_start'  => '<tr>',
    'row_end'    => '</tr>',
    'cell_start' => '<td>',
    'cell_end'   => '</td>',

    'row_alt_start'  => '<tr>',
    'row_alt_end'    => '</tr>',
    'cell_alt_start' => '<td>',
    'cell_alt_end'   => '</td>',

    'table_close' => '</table>',
];

$table->setTemplate($template);

Catatan

Anda akan melihat ada dua set blok "baris" di template. Ini memungkinkan Anda untuk membuat warna baris bergantian atau elemen desain yang bergantian dengan setiap iterasi data baris.

Anda TIDAK diharuskan untuk mengirimkan template lengkap. Jika Anda hanya perlu mengubah bagian dari layout, Anda cukup mengirimkan elemen-elemen tersebut. Dalam contoh ini, hanya tag pembuka tabel yang diubah:

<?php

$template = [
    'table_open' => '<table border="1" cellpadding="2" cellspacing="1" class="mytable">',
];

$table->setTemplate($template);

Anda juga dapat mengatur default untuk ini dengan meneruskan array pengaturan template ke konstruktor Table:

<?php

$customSettings = [
    'table_open' => '<table border="1" cellpadding="2" cellspacing="1" class="mytable">',
];

$table = new \CodeIgniter\View\Table($customSettings);

Menyinkronkan Baris dengan Judul

Baru pada versi 4.4.0.

Metode setSyncRowsWithHeading(true) memungkinkan setiap nilai data ditempatkan di kolom yang sama seperti yang didefinisikan dalam setHeading() jika array asosiatif digunakan sebagai parameter. Ini sangat berguna saat menangani data yang dimuat melalui REST API di mana urutannya tidak sesuai keinginan Anda, atau jika API mengembalikan terlalu banyak data.

Jika baris data berisi kunci yang tidak ada di judul, nilainya difilter. Sebaliknya, jika baris data tidak memiliki kunci yang tercantum di judul, sel kosong akan ditempatkan di tempatnya.

<?php

$table = new \CodeIgniter\View\Table();

$table->setHeading(['name' => 'Name', 'color' => 'Color', 'size' => 'Size'])
    ->setSyncRowsWithHeading(true)
    ->addRow(['color' => 'Blue', 'name' => 'Fred', 'size' => 'Small'])
    ->addRow(['size' => 'Large', 'age' => '24', 'name' => 'Mary'])
    ->addRow(['color' => 'Green']);

echo $table->generate();
?>

<!-- Generates a table with this prototype: -->
<table border="0" cellpadding="4" cellspacing="0">
    <thead>
        <tr>
            <th>Name</th>
            <th>Color</th>
            <th>Size</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>Fred</td>
            <td>Blue</td>
            <td>Small</td>
        </tr>
        <tr>
            <td>Mary</td>
            <td></td>
            <td>Large</td>
        </tr>
        <tr>
            <td></td>
            <td>Green</td>
            <td></td>
        </tr>
    </tbody>
</table>

Penting

Anda harus memanggil setSyncRowsWithHeading(true) dan setHeading([...]) sebelum menambahkan baris apa pun melalui addRow([...]) di mana penataan ulang kolom terjadi.

Menggunakan array sebagai input untuk generate() menghasilkan hasil yang sama:

<?php

$data = [
    [
        'color' => 'Blue',
        'name'  => 'Fred',
        'size'  => 'Small',
    ],
    [
        'size' => 'Large',
        'age'  => '24',
        'name' => 'Mary',
    ],
    [
        'color' => 'Green',
    ],
];

$table = new \CodeIgniter\View\Table();

$table->setHeading(['name' => 'Name', 'color' => 'Color', 'size' => 'Size'])
    ->setSyncRowsWithHeading(true);

echo $table->generate($data);

Referensi Class

class CodeIgniter\View\Table
$function = null

Memungkinkan Anda untuk menentukan fungsi PHP native atau objek array fungsi yang valid untuk diterapkan ke semua data cell.

<?php

$table = new \CodeIgniter\View\Table();

$table->setHeading('Name', 'Color', 'Size');
$table->addRow('Fred', '<strong>Blue</strong>', 'Small');

$table->function = 'htmlspecialchars';
echo $table->generate();

Dalam contoh di atas, semua data cell akan dijalankan melalui fungsi htmlspecialchars() PHP, menghasilkan:

<td>Fred</td><td>&lt;strong&gt;Blue&lt;/strong&gt;</td><td>Small</td>

addRow([$args = [][, ...]])
Parameter:

  • $args (mixed) -- Sebuah array atau beberapa string yang berisi nilai baris

Kembali:

Instance Table (method chaining)

Return type:

Table

Memungkinkan Anda untuk menambahkan baris ke tabel Anda. Anda dapat mengirimkan array atau parameter diskrit:

<?php

$table->addRow('Blue', 'Red', 'Green'); // or

$table->addRow(['Blue', 'Red', 'Green']);

Jika Anda ingin mengatur atribut tag cell individual, Anda dapat menggunakan array asosiatif untuk cell tersebut. Kunci asosiatif data mendefinisikan data cell. Pasangan kunci => nilai lainnya ditambahkan sebagai atribut kunci='nilai' ke tag:

<?php

$cell = ['data' => 'Blue', 'class' => 'highlight', 'colspan' => 2];
$table->addRow($cell, 'Red', 'Green');

?>

<!-- Generates: -->
<td class='highlight' colspan='2'>Blue</td><td>Red</td><td>Green</td>
makeColumns([$array = [][, $columnLimit = 0]])
Parameter:

  • $array (array) -- Sebuah array yang berisi data beberapa baris

  • $columnLimit (int) -- Jumlah kolom dalam tabel

Kembali:

Sebuah array kolom tabel HTML

Return type:

array

Metode ini mengambil array satu dimensi sebagai input dan membuat array multi-dimensi dengan kedalaman yang sama dengan jumlah kolom yang diinginkan. Ini memungkinkan satu array dengan banyak elemen untuk ditampilkan dalam tabel yang memiliki jumlah kolom tetap. Pertimbangkan contoh ini:

<?php

$list = ['one', 'two', 'three', 'four', 'five', 'six', 'seven', 'eight', 'nine', 'ten', 'eleven', 'twelve'];

$newList = $table->makeColumns($list, 3);

$table->generate($newList);

?>

<!-- Generates a table with this prototype: -->
<table border="0" cellpadding="4" cellspacing="0">
    <tr>
        <td>one</td>
        <td>two</td>
        <td>three</td>
    </tr>
    <tr>
        <td>four</td>
        <td>five</td>
        <td>six</td>
    </tr>
    <tr>
        <td>seven</td>
        <td>eight</td>
        <td>nine</td>
    </tr>
    <tr>

clear()
Kembali:

Instance Table (method chaining)

Return type:

Table

Memungkinkan Anda untuk menghapus judul tabel, data baris, dan keterangan. Jika Anda perlu menampilkan beberapa tabel dengan data yang berbeda, Anda harus memanggil metode ini setelah setiap tabel telah dihasilkan untuk menghapus informasi tabel sebelumnya.

Contoh

<?php

$table = new \CodeIgniter\View\Table();

$table->setCaption('Preferences')
    ->setHeading('Name', 'Color', 'Size')
    ->addRow('Fred', 'Blue', 'Small')
    ->addRow('Mary', 'Red', 'Large')
    ->addRow('John', 'Green', 'Medium');

echo $table->generate();

$table->clear();

$table->setCaption('Shipping')
    ->setHeading('Name', 'Day', 'Delivery')
    ->addRow('Fred', 'Wednesday', 'Express')
    ->addRow('Mary', 'Monday', 'Air')
    ->addRow('John', 'Saturday', 'Overnight');

echo $table->generate();
setSyncRowsWithHeading(bool $orderByKey)
Kembali:

Instance Table (method chaining)

Table

Mengaktifkan setiap kunci data baris untuk diurutkan berdasarkan kunci judul. Ini memberikan kontrol lebih atas data yang ditempatkan di kolom yang benar. Pastikan untuk mengatur nilai ini sebelum memanggil metode addRow() pertama.
